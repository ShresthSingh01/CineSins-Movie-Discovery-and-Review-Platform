## üõ°Ô∏è Focus through Rationalization: What we removed & why

To maintain a clean, analytical, data-first privacy-focused aesthetic, the following features were intentionally removed or disabled:

* **Removed `sceneTags`**: Manual taxonomy introduces friction. We use automated emotional analytics derived from metadata and reviews.
* **Removed `watchNow`**: Deep-linking to streaming vendors turns an analysis tool into a consumption app. We focus on pre-watch analysis and post-watch insight.
* **Removed `compatibility`**: Personality is personal. We've pivoted from social gimmicks to single-user deep-dive analytics.
* **Removed `neonUI`**: Sprawling animations and "Spin Again" gamification distract from data-first movie intelligence.
